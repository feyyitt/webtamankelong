📋 Fitur Sistem Admin

📊 Dashboard Statistik

Melihat ringkasan total reservasi, jumlah yang menunggu approval, disetujui, dan dibatalkan dalam kartu statistik yang informatif.

🔍 Filter & Pencarian

Filter reservasi berdasarkan status, rentang tanggal kunjungan, dan pencarian berdasarkan nama pemesan.

✅ Manajemen Status

Approve atau tolak reservasi yang masuk. Status akan otomatis ter-update dan dapat dilihat dalam tabel.

👁️ Detail Reservasi

Lihat detail lengkap setiap reservasi termasuk makanan yang dipesan, total harga, dan catatan khusus.

🗑️ Hapus Data

Hapus reservasi yang tidak diperlukan dengan konfirmasi untuk mencegah penghapusan tidak sengaja.

📥 Export Data

Export data reservasi ke format CSV untuk analisis lebih lanjut atau backup data.

🚀 Cara Menggunakan Sistem

1Akses Admin

Buka halaman admin dengan mengklik ikon ⚙️ di footer website utama atau langsung ke admin_login.html

2Login

Masukkan username dan password admin, kemudian klik "Masuk ke Dashboard"

3Monitor Reservasi

Dashboard akan menampilkan:

  • Statistik reservasi dalam kartu yang mudah dibaca
  • Tabel semua reservasi dengan informasi lengkap
  • Status setiap reservasi (Menunggu/Disetujui/Dibatalkan)

4Kelola Reservasi

Untuk setiap reservasi, Anda dapat:

  • Lihat Detail: Klik tombol "Lihat" untuk melihat informasi lengkap
  • Setujui: Klik "Setujui" untuk meng-approve reservasi
  • Tolak: Klik "Tolak" untuk menolak reservasi
  • Hapus: Klik "Hapus" untuk menghapus data reservasi

5Filter & Cari

Gunakan filter untuk:

  • Menampilkan reservasi berdasarkan status tertentu
  • Melihat reservasi dalam rentang tanggal tertentu
  • Mencari reservasi berdasarkan nama pemesan

6Export Data

Klik "Export CSV" untuk mengunduh data reservasi yang sudah difilter dalam format spreadsheet

💾 Penyimpanan Data

Sistem ini menggunakan localStorage browser untuk menyimpan data reservasi. Data akan tersimpan di browser admin dan dapat diakses selama browser tidak di-clear cache-nya.

📝 Catatan Penting:

  • Data disimpan di localStorage browser, pastikan tidak menghapus cache browser
  • Untuk production, disarankan menggunakan database server
  • Backup data secara berkala menggunakan fitur Export CSV
  • Session admin berlaku 24 jam, setelah itu harus login ulang

🔄 Integrasi dengan Form Reservasi

Sistem admin sudah terintegrasi otomatis dengan form reservasi. Setiap kali ada reservasi baru dari website:

  • Data otomatis tersimpan di sistem admin dengan status "Menunggu"
  • Admin akan melihat reservasi baru di dashboard
  • ID unik otomatis dibuat untuk setiap reservasi
  • Customer akan mendapat notifikasi bahwa reservasi telah diterima
  • Admin dapat menghubungi customer melalui nomor WhatsApp yang tersimpan

📱 Responsive Design

Dashboard admin sudah responsive dan dapat diakses dari:

  • Desktop/Laptop - Tampilan penuh dengan semua fitur
  • Tablet - Layout yang disesuaikan untuk layar sedang
  • Mobile - Interface yang dioptimalkan untuk smartphone
